# Django URL Shortener

URL Shortener app with Django.
Django 1.6.11 
Python 2.7.6
Mongodb 2.4.9

--------------------------------

Running this app (terminal):

$ python manage.py syncdb 
$ python manage.py runserver

--------------------------------

To check the db entries (terminal):

start mongo cli -> $ mongo
$ use urlshortner
$ db.mainApp_urls.find()
